# aillium-remote

## Purpose
`aillium-remote` manages **secure remote access and device connectivity** for Aillium.

It is built around MeshCentral.

## What This Repo Does
- MeshCentral configuration
- Agent packaging
- Device grouping & policies
- Session access rules

## What This Repo Does NOT Do
- No UI automation
- No AI logic
- No planning or decision-making

## Architecture Role
Remote Access & Device Layer

## Security Notes
- Explicit consent required
- Session logging enabled
- Access revocable at any time
